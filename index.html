Privacy policy for n8n Blicki

Last updated: June 23, 2025

This application accesses your Gmail account solely to automate email processing and workflow management through n8n automation.

Data collection and use

  • We access only the Gmail data necessary for the application's functionality
  • No personal data is stored permanently on external servers
  • Data is processed locally through your n8n instance
  • No data is shared with third parties
  • Email content is only accessed to perform the automated workflows you configure

Data security

  • All data transmission uses secure protocols (HTTPS/TLS)
  • Access is limited to the authenticated user only
  • No data is retained beyond what's necessary for operation
  • OAuth tokens are stored securely within your n8n instance

Scope of access

This application may request access to:

  • Read your email messages and metadata
  • Send emails on your behalf (if configured)
  • Manage labels and folders (if configured)

Access is used exclusively for the automation workflows you create and configure.

Data retention

No email data is permanently stored by this application. Data is processed in real-time for workflow execution and discarded immediately after processing.
